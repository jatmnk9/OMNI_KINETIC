# **App Name**: Omni-Scent Hub

## Core Features:

- Omni-Kinetic Welcome Hub: Upon app launch, users view an ecosystem overview in a minimalist black and white palette, learning about kinetic perfumery before device linking.
- Kinetic Device Pairing: Scan for nearby kinetic perfumery devices via BLE/NFC and display a high-quality 3D render of the detected model (Apex, Synapse, Kinetic) for user confirmation.
- Dynamic Brand Theming: Transition the app's visual theme, color palette, and graphics based on the connected device's brand (Prada, YSL, Biotherm), while generic sections remain in the black and white base.
- AI-Powered Scent Boutique: Browse and purchase Starter Kits and individual fragrance cartridges. An AI tool suggests personalized scent recommendations based on user interaction and biometric data, and enables management of subscription plans (Base, Essential, Premium/Elite).
- Biometric Scent Activation Engine: Process specific telemetry data from the connected device (e.g., accelerometer, GSR, aquatic metrics) and, when predefined biometric thresholds are met, instruct the piezoelectric actuator to release precise micro-doses of fragrance.
- Cartridge & Omni-Rewards Tracker: Track remaining fragrance levels and cycle days, managing predictive refills for subscription plans. The system also credits points for returning empty cartridges and achieving usage milestones, unlocking exclusive aromas.

## Style Guidelines:

- Background Color: 'Very Dark Gray' (#121214). This provides a luxurious, tech-minimalist foundation that allows dynamic themes to stand out against the initial black and white scheme.
- Primary Color: 'Deep Charcoal' (#242429). Used for key UI elements and text against the dark background, offering subtle contrast and sophistication.
- Accent Color: 'Vibrant Cyan-Blue' (#3399CC). Used for interactive elements, highlights, and data visualization, adding a modern, energetic touch that complements the dark base and aligns with the tech focus.
- Brand Theme (Prada): Apply a rich '#3D0A09' (dark crimson) palette with silver and gold accents for Apex Essence device screens, reflecting its technical aesthetic.
- Brand Theme (YSL): Utilize a deep '#301540' (dark violet) palette with golden accents and a sophisticated night design for Synapse device dashboards and metrics.
- Brand Theme (Biotherm): Implement a refreshing '#0097A9' (cyan blue) palette with silver details and organic data visuals for Kinetic device data visualization.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable design, adaptable across varying weights for different hierarchical elements and data displays.
- Minimalist, vector-based line icons with a premium, sleek aesthetic to complement the clean design. Use realistic 3D renders for accurate device representation and interactive elements.
- Emphasize immersive visuals, particularly high-quality 3D device renders and interactive data visualizations within modular dashboard layouts specific to each device. Prioritize clear hierarchy and responsive design.
- Smooth, subtle transitions for dynamic theme changes ('skin' mutations) and elegant micro-interactions to create a fluid user experience, especially during device scanning and onboarding.
